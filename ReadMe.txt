2. PREREQUISITES
================
 -> Windows 10 
 -> System RAM >= 16.0 GB
 -> Python - 3.4.3
 -> Python package management system(pip) should be installed in the system.
	pip3 install selenium

Execution Step
--------------
1. Change to CodingRound folder
2. Update the driver details and Web site to test detail in Config.xml file
3. Open the command line and execute the below command
	python Driver.py -i E:\MyData\CodingRound\Config.xml -t flight
	python Driver.py -i E:\MyData\CodingRound\Config.xml -t Hotel
	python Driver.py -i E:\MyData\CodingRound\Config.xml -t SIGNIN
